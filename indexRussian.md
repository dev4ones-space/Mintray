# Mintray
Mintray — прокси-клиент для macOS и Linux. Для вашей приватности и свободы. Без зависимостей, только TUI. _(работает даже на голых серверах без окружения рабочего стола, отлично через SSH)_

## Почему Mintray
- **Без зависимостей.** _(Работает без КАКИХ-ЛИБО зависимостей на голой машине Linux и macOS в полном объёме)_
- **Работает одинаково на macOS и Linux** _(не нужно изучать отдельное приложение под каждую платформу)_
- **Режим TUN.** _(пропускает через прокси весь трафик, кроме локального)_
- **Лёгкий** _(без тяжёлого фонового сервиса, просто Python3)_
- **Простой дизайн** _(чистый, простой терминальный интерфейс, понятный каждому)_

## Прочая информация
1. UDP может работать некорректно — особенность ядра Xray _(мы не можем это исправить или как-то повлиять на это, поэтому всё, что использует UDP, может просто не работать, например звонки в WhatsApp или WebRTC в целом)_
2. Windows не поддерживается. _(1. Нет встроенной stdlib — нет curses (это основной интерфейс, он обязателен). 2. Слишком строгие ограничения (например, отсутствует полноценная реализация root). 3. Проблемы совместимости сети — потребовалась бы отдельная реализация всего сетевого стека (macOS и Linux в целом безопасно совместимы друг с другом))_
3. Возможно, не запустится на Linux — всё достаточно протестировано, и отмечено, что приложение полностью работоспособно на Linux и macOS, но у Linux слишком много семейств дистрибутивов, и мы не можем оптимизировать приложение под каждый из них. Вот список подтверждённо рабочих ОС:
- macOS _(100% совместимость с Ventura и новее, обновления никогда не затрагивают то, что может что-то сломать)_
- CachyOS и Arch _(дистрибутив Linux; дистрибутивы на основе Arch Linux, скорее всего, тоже будут работать без проблем)_
## Сборка из исходного кода
```bash
git clone https://github.com/dev4ones-space/Mintray.git
cd Mintray
```
Скачайте последний релиз бинарников из следующих репозиториев
[XTLS/Xray-core](https://github.com/XTLS/Xray-core/releases)
[xjasonlyu/tun2socks](https://github.com/xjasonlyu/tun2socks/releases)
```bash
pip install pyinstaller
mkdir bin && cp /path/to/xray /path/to/tun2socks bin/
pyinstaller Mintray.spec
```
## Сборка из исходного кода (пошагово)
_(для всего этого требуется, чтобы Python3 был установлен и находился в вашем `$PATH`, скачать установщик можно на [python.org](https://www.python.org/downloads/))_
1. Клонируйте репозиторий и перейдите в него: _(`cd` меняет директорию, в которой сейчас находится ваш терминал)_
```bash
git clone https://github.com/dev4ones-space/Mintray.git && cd Mintray
```
2. Соберите бинарники для вашего устройства: _(укажите архитектуру устройства (x84_64, aarch64/arm) и ОС (darwin для macOS или просто linux))_
- **[XTLS/Xray-core](https://github.com/XTLS/Xray-core/releases)**
- **[xjasonlyu/tun2socks](https://github.com/xjasonlyu/tun2socks/releases)**
3. Установите модуль PyInstaller для Python: _(требует, чтобы pip был установлен и корректно работал)_
```bash
pip install pyinstaller
```
или
```bash
python3 -m pip install pyinstaller
```
_(также, если установка не удастся из-за ошибки externally managed, добавьте к команде этот аргумент: `--break-system-packages`)_
4. Создайте директорию `bin/` и поместите в неё бинарники: _(требует небольшого изменения команды с указанием своего пути)_
```bash
mkdir bin && cp /path/to/xray /path/to/tun2socks bin/
```
5. Соберите бинарник:
```
pyinstaller Mintray.spec
```
или
```
python3 -m PyInstaller Mintray.spec
```
### После всего этого полностью рабочий исполняемый файл должен быть собран в `dist/Mintray`
